# rednighted.vim

rednighted.vim is a dark colorscheme with red tones.
Originally made by [wolf-dog](https://github.com/wolf-dog), I simply recolored it to match my desktop theme.
Forked from [here](https://github.com/wolf-dog/nighted.vim)

Designed for 256-color terminal emulaters, and also works well on gui version.

## Screenshot

![](https://files.catbox.moe/23ppbh.png)

## Installation

Copy rednighted.vim to your .vim/colors directory, then do

    :colorscheme rednighted

## Configurations

If you prefer transparent backgrounds, add the following to your .vimrc.

    let g:rednighted_transparent_bg = 1
    " Then load normally
    colorscheme rednighted
